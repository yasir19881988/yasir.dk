The eLabel module/plugin is an extension of the SaaS platform. Store owners on PrestaShop, Magento, WordPress/WooCommerce, and Shopify can install the module or plugin and, through an API key, automatically display product declarations on the product page using the entered EAN number. Merchants can also control and sort which declaration data points are shown on each product page.
