B2BCANDY

Description

  • Yasir Rasool

  • E-Commerce

  • 01.07.2023

B2BCANDY is a B2B e-commerce implementation for Denmark and the wider EU market, with dedicated pricing models for customer groups, DKK and EUR handling, and backend order flows designed for wholesale operations.

Technology

  • PrestaShop
  • PHP
  • MySQL
  • B2B Pricing Rules
  • Multi-Currency (DKK/EUR)
  • REST API Integrations
